# Student Attendance Management System

A ReactJS-based Attendance Management System.

## Features
- Add Students
- Delete Students
- Search Students
- Mark Present/Absent
- Attendance Percentage
- Dashboard Analytics
- Dark/Light Mode
- Reports Module
- Local Storage

## Technologies Used
- ReactJS
- React Router DOM
- CSS3
- JavaScript
- Local Storage
